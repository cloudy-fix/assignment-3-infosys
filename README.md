# Infosys Compliance Checker V2

Continuation of the Infosys compliance checker work, focused on document analysis, LLM-assisted review, and risk-focused contract interpretation.

## Interview Angle

This project is a good follow-up discussion point after the first compliance checker assignment. Use it to explain iteration: improving document review structure, refining outputs, and thinking through reviewer-friendly compliance findings.

- How the second version improves the first assignment
- Document analysis workflow design
- Compliance result formatting
- LLM/RAG limitations and validation needs
- Practical risk review use cases
